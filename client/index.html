=========================== JavaScript Tree Coding Test ===========================

Render a tree using jQuery / JavaScript client-side in the browser.

Clicking on a node should call a hook that will display the "CODEVal" of that node. This should not fire when expanding / collapsing the tree.
1. The array is in the format it would be received from the database - an array of rows
2. If CODEParentID is null, that means it is a root node. Otherwise, CODEParentID has the CODEID of the parent element in the tree
3. CODEIcon is the suffix of the icon used to render that node - "icon_xxxxx.png"
4. CODEVal can be ignored
5. CODETxt is the Display Text
